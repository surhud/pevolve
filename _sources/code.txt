Cosmology code documentation
============================

All the examples assume that you have set the PYTHONPATH environment variable
to the installation location of pevolve, and the PATH to the pevolve directory.
Run the following commands from inside the pevolve directory

.. sourcecode:: bash

    $ export PYTHONPATH=$PYTHONPATH:`pwd`/install/lib/python2.7/site-packages
    $ export PATH=$PATH:`pwd`
