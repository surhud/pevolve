Installation¶

The pseudo-evolution package depends upon the following
  1. swig
  2. GNU scientific library
  3. The public mandc code by Donghai Zhao
  4. Any of the following fortran compilers: (gfortran, g77, f77, ifort)
Optional
  • Sphinx for documentation

The installer can install the first three dependencies if the computer has access to the internet and at least python (and its header files) is installed. On Ubuntu install most of the dependencies using:

$ sudo apt-get install python python-dev python-sphinx swig libgsl0ldbl libgsl0-dev gfortran

On Fedora linux:

$ sudo yum install python python-devel python-sphinx swig gsl gcc-gfortran

On a mac, if you have homebrew,

$ brew install python swig gsl gfortran

Step by step instructions¶

Download source code
  • Source code for pevolve
$ git clone http://github.com/surhud/pevolve
$ cd pevolve
$ chmod a+x setup_requirements.sh
$ ./setup_requirements.sh

If all goes well, then pevolve should be installed in the install directory inside pevolve. One final step is to check the instructions.txt file generated by the installer. It suggests modifications to your ~/.bashrc file so that the new executables and files are in your path. Some of the common cases are:

Add the directory where pevolve was installed to your PYTHONPATH variable.

$ export PYTHONPATH=$PYTHONPATH:`pwd`/install/lib/python2.7/site-packages
$ export PATH=$PATH:`pwd`/install/bin

Please modify the python path in the code above depending upon your system install. Check instructions.txt to see the path to pevolve for example.

If the installer had to install gsl on your system then instructions.txt will ask you to add the path where gsl was installed to your LD_LIBRARY_PATH environment variable.

$ # Only if gsl had to be installed by the installer
$ export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:`pwd`/install/lib/

All of these instructions can be found in instructions.txt file generated by the installer. Once the instructions in the file instructions.txt are followed, the code should be setup for use! Please see the cosmology documentation for further examples.
